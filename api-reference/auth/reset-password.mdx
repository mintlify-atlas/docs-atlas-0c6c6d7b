---
title: "Reset Password"
description: "Reset a user's password using the OTP received by email."
---

## Overview

Completes the password reset flow by verifying the OTP sent to the user's email and setting a new password. Call [Forgot Password](/api-reference/auth/forgot-password) first to obtain the OTP.

## Endpoint

```
POST /api/auth/reset-password
```

## Request

### Headers

| Header | Value |
|---|---|
| Content-Type | application/json |

### Body parameters

<ParamField body="email" type="string" required>
  The email address associated with the account.
</ParamField>

<ParamField body="otp" type="string" required>
  The one-time password received at the user's email address.
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password to set for the account.
</ParamField>

## Response

### Success

<ResponseField name="...data" type="object">
  Confirmation message returned by the server indicating the password was reset.
</ResponseField>

### Error

<ResponseField name="error" type="string">
  Human-readable error message describing why the reset failed.
</ResponseField>

<Note>
  You must first request an OTP via [Forgot Password](/api-reference/auth/forgot-password) before calling this endpoint. Submit the OTP exactly as received in the email.
</Note>

## Example

```typescript TypeScript
import { SERVER_ENDPOINT } from "@env";

const API = SERVER_ENDPOINT ?? "";

const resetPassword = async (
  email: string,
  otp: string,
  newPassword: string,
) => {
  const res = await fetch(`${API}/api/auth/reset-password`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      email,
      otp,
      newPassword,
    }),
  });

  const data = await res.json();

  if (!res.ok) {
    throw new Error(data.error || "Reset password request failed");
  }

  return data;
};
```
